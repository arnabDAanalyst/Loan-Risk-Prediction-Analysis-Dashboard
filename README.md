# LOAN RISK PREDICTION

The Loan Risk Prediction Dataset is a fully synthetic dataset designed to simulate a real-world financial lending environment. It was created specifically for practicing end-to-end machine learning workflows, including data preprocessing, feature engineering, model training, and performance evaluation. The primary objective of this dataset is to predict whether a loan application will be approved based on applicant demographic, financial, and employment-related information.
